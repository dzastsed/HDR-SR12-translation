# HDR-SR12-translation
translation and language switching project for certain sony camcorder

This is a project which main goal is to swap language from Japanese to English for JDM HDR-SR11 or SR12 camcorders. More models might be supported, but for now only this one is.

Another goal of this repo is to make documented spreadsheets for easier translation to other languages (my goal is to make Lithuanian translation).
